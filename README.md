# fish-admin
> fish-admin is simple development boilerplate <br/>
> ui: based on fish-ui (vue 2) <br/>
> api: based on springboot (java)

# Quick Start
``` bash
$ git clone https://github.com/myliang/fish-admin.git
# 1 api
$ cd fish-admin/api
$ gradle bootRun

# 2 ui
$ cd fish-admin/ui
$ npm install
$ npm run dev
```

# Functional module

1. user management
2. role management
3. change password
4. login
5. exit
